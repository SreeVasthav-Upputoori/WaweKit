Key Ideas

RDKit concepts
  • rdMolStandardize.Cleanup — one true representation
  • LargestFragmentChooser — the parent molecule
  • Uncharger — sensible neutral form
  • TautomerEnumerator.Canonicalize — merge tautomers
  • reuse heavy helper objects across the whole set (perf)
Software concepts
  • Immutable in, new out — inputs never mutated; failures keep the original
  • Report, not just results — before/after + steps per molecule
  • Frozen options dataclass — serializable to Settings/batch later
  • Reused FunctionWorker — the generic worker pays off again
  • Static factory dialog — get_options() → Options | None
