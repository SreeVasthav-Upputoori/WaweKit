The demo set: 15 molecules, grouped

RepresentationDistinct scaffoldsWhy

Acyclic molecules (ethanol, hexane, acids) have no scaffold — surfaced honestly as one (acyclic) group, never crashed on.

Key Ideas

RDKit / chemistry
  • GetScaffoldForMol — exact Murcko core
  • MakeScaffoldGeneric — all-carbon, single-bond graph
  • Acyclic = empty scaffold — the edge case that matters
  • Diversity = distinct scaffolds ÷ molecules
Software design
  • Model in models, compute in services — 5th proof of the seam
  • Cache in place — repaint one column, keep scroll/selection
  • Panel emits intent — no worker, no RDKit in the widget
  • 2nd AND-ed filter channel — scaffold click ∧ text query
  • View toggle, not compute option — switch with no recompute
