The new rule: 3D lives beside the record, not on it

record.molstays 2D — table thumbnail & Structure panel depict this
record.conformers → ConformerSet.mol_3da SEPARATE mol: +Hs, many conformers, all the 3D geometry

Every earlier derived result was a small cached value; 3D is a whole molecule, so it points to a distinct object rather than overwriting the 2D identity.

Key Ideas

Chemistry / robustness
  • Seeded ETKDG — reproducible runs (research-grade)
  • MMFF→UFF fallback — never fail on missing params; record the FF actually used
  • RMSD pruning — kept conformers are genuinely distinct shapes
  • SDF export — honest handoff to PyMOL / docking
Software design
  • Cache-in-place + report — 6th module on the same seam
  • Runs on selection first — expensive default earns the refinement
  • Panel = pure consumer — follows selection, emits nothing
  • Web-feature testing split — model/service unit-tested; 3D screenshot-verified (async WebGL)
