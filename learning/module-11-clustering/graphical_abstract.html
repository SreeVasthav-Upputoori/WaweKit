Two algorithms

MethodIdeaNeedsRole
Butinasphere exclusion on Tanimoto distancea cutoff (not K)the cheminformatics default

Both numbered largest-first (cluster 0 = biggest) — a stable convention the sort and the palette rely on. Demo: alkane family → cluster 0.

Key Ideas

Design
  • Cluster id is relational → cached like similarity, carries its ClusterRun; stale ids actively cleared
  • Cached (not result-object) because two surfaces read it: column + map
  • Lazy sklearn import; Butina is pure RDKit
  • Reuse: fingerprints, FingerprintOptionsWidget (4th consumer), FunctionWorker (8th job)
The Module 10 payoff — categorical colour
tab20 by cluster id, no colourbar
  • Continuous colour-by was viridis + colourbar
  • A cluster is a category, not a scale → qualitative palette
  • Auto-selected after a run → families light up on the map
  • Missing = grey, as everywhere
