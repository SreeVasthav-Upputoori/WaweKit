The finding R1 already surfaced (a failing test taught us)

Canonical-tautomer rewrote 2-hydroxypyridine's SMILES Oc1ccccn1 → O=c1cccc[nH]1 — but its InChIKey did not change. InChI normalizes tautomers internally.

⟹ The identity key changes what "reproducible" means. SMILES reveals the divergence; InChIKey masks it.

The two keys disagree on which molecules are pipeline-dependent — a non-obvious result no standardization tool reports. The study measures both.

Design

  • StandardForm carries both identities (SMILES + InChIKey) — so R2 can measure agreement on either
  • Never mutates input · reuses costly helpers (lru_cache) · never aborts on one bad molecule
  • Qt-free → runs in the benchmark harness (R5) and a notebook
