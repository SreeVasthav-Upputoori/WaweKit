The two-way link (and the loop it avoids)

Plot: click / lasso → points_selected
→ table.select_records → Structure & Conformer panels follow
Table selection → highlight_records (rings points)
no signal emitted → no cycle

The "display" path (highlight, redraws in place, keeps zoom) is kept strictly separate from the "intent" path (emit) — that separation is what makes bidirectional linking safe.

Key Ideas

Method / science
  • PCA — fast, deterministic, axes report variance ("46% shown")
  • t-SNE — clusters separate; axes unitless; perplexity clamped to n
  • Colour-by — any descriptor or similarity; missing = grey, not zero
  • Skipped molecules reported, never silently dropped
Software design
  • Projection is relational → lives in the service, never cached on records
  • Lazy sklearn import — no startup cost until first projection
  • Matplotlib toolbar = free PNG/PDF/SVG export for papers
  • matplotlib + sklearn promoted to core deps (Modules 10–11 need them)
