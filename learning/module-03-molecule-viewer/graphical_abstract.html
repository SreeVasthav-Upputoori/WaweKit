Wawekit · Module 03

Molecule Viewer — Making Chemistry Visible

2D structure thumbnails in every table row + a selection-driven Structure panel, all theme-aware and vector-crisp.

Rendering Flow (one renderer, two consumers)

services/rendering · render_svg(mol, w, h, dark, legend) pure RDKit → SVG string · copies mol (never mutates records) · Compute2DCoords when SMILES-born · SetDarkMode palette · transparent background
▲ same function will feed PDF reports (Module 14)  ·  consumed by ▼
StructureDelegate QStyledItemDelegate paints thumbnails in the table · pixmap cache keyed by SMILES+theme · 2× HiDPI · only visible cells render
StructureViewerPanel QSvgWidget (vector = crisp at any size) · name/formula/SMILES · SDF properties table · Copy SMILES · Save SVG/PNG
selection: table selection_changed(record) → viewer  ·  theme: ThemeManager.theme_changed → both re-render

Key Ideas

RDKit concepts
  • 2D coordinates — SMILES-born mols have none; rdDepictor.Compute2DCoords lays them out
  • PrepareAndDrawMolecule — kekulization + wedges done right
  • SetDarkMode — black bonds are invisible on dark themes
  • Draw a copy — records are shared; renderers must not mutate
Qt concepts
  • QStyledItemDelegate — custom painting inside Model/View cells
  • Custom data roles — RECORD_ROLE hands the domain object to the delegate
  • QSvgWidget / QSvgRenderer — vector display + rasterized export
  • Observer pattern — ThemeManager became a QObject with a signal
  • toggleViewAction() — free, self-syncing dock show/hide actions

A Real Bug the Tests Caught

Reverse-order table — setSortingEnabled(True) immediately applies the header's default sort indicator, which is column 0 descending. Every loaded file would have displayed upside-down. Caught by the selection-signal test; fixed with an explicit sortByColumn(0, AscendingOrder). Lesson: tests that assert on which object comes back — not just that something comes back — catch ordering bugs.

Engineering Decisions

renderer in services, not gui (reused for reports) render-on-demand + cache (100k rows stay smooth) SVG everywhere; rasterize only at export panel translates Qt selection → domain object depiction failure logs, never crashes painting verified: 31 tests · ruff · black
