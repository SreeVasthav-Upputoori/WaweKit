Two Decisions Worth Remembering

One typed field, not eight
  • descriptors: DescriptorSet | None — one slot per concept, so the record doesn't grow per value as Modules 6+ pile on
  • Staleness solves itself — standardization builds new records → None again → recomputed
  • Mutated in place (Module 4 returned new records) because a descriptor changes nothing — it's a cache, like .smiles
  • DESCRIPTOR_SPECS registry drives headers, cells, sort keys, tooltips and filter tokens — new descriptor = one line
Quick-filter: not regex
  • Regex matches display text → can't compare numbers. So the proxy asks the record, seeing real floats
  • aspirin → substring · MW < 500 → numeric · Lipinski = 0 → pass filter
  • Invalid ≠ empty — MW > shows "⚠ Incomplete comparison", never a silently blank table
  • Uncomputed rows are hidden by numeric filters — "unknown" isn't "matches"
